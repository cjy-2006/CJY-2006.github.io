十、项目总结

该脚本实现了 Markdown → Hexo → GitHub Pages 的一键自动化发布流程，将原本需要手动执行的多个命令整合到一个 Python 程序中。

整个开发过程中主要练习了以下能力：

  • 函数封装与模块划分
  • Python 文件读写
  • subprocess 调用系统命令
  • Git 自动化操作
  • Hexo 静态博客生成
  • 命令执行状态判断与失败重试
  • 自动化脚本设计思路

虽然这是一个轻量级脚本，但已经具备了完整自动化工具的基本结构，为后续开发更复杂的工程项目（如增加配置、日志、GUI、云端部署等）打下了基础。
