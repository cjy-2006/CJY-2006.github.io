md文档基本用法

md文档基本用法

你想了 Markdown 文档的核心基本用法，以便能快速上手编写规范、易读的 Markdown 格式文档。Markdown 是一种轻量级标记语言，语法简单易记，能让你用纯文本快速排版出结构化的文档，下面我会把最常用的基础用法逐一讲清楚。

一、核心基础用法（附语法 + 示例）

1. 标题

用于定义文档的层级标题，最多支持 6 级，语法是「# + 空格 + 标题内容」，# 的数量对应标题级别。

markdown

效果：一级标题字号最大，逐级递减，视觉上有明显的层级区分。

2. 文本格式（粗体 / 斜体 / 删除线）

用于突出文本重点，语法简单且常用：

markdown

示例：

  • 这是斜体
  • 这是粗体
  • 这是粗斜体
  • 这是删除线

3. 列表（有序 / 无序列表）

（1）无序列表

用 -、+ 或 * 加空格表示，三种符号效果一致：

markdown

（2）有序列表

用「数字 + . + 空格」表示，数字顺序不影响渲染效果（最终会按 1、2、3 排序）：

markdown

4. 链接与图片

（1）链接

有两种常用形式：行内式和参考式（适合链接较多的文档）：

markdown

（2）图片

语法和链接类似，只是开头多一个 !，支持本地图片和网络图片：

markdow

5. 代码块（单行 / 多行）

（1）单行代码

用反引号 ``` 包裹：

markdow

（2）多行代码

用三个反引号 ``` 包裹，开头可指定编程语言（实现语法高亮）：

markdown

plaintext

7. 分割线

用三个及以上的 -、* 或 _ 表示（符号之间不能有其他字符，可空格）：

markdown

8. 表格

用竖线 | 分隔列，第二行用 - 定义表头和内容的分隔（:--- 左对齐、---: 右对齐、:---: 居中）：

markdown

效果：姓名左对齐，年龄居中，职业右对齐。

二、其他常用小功能

  • 换行：行尾加 2 个空格，或直接空一行；

  • 转义字符：用 \ 转义 Markdown 特殊符号（如 \# 会显示 # 而不是标题）；

  • 任务列表：

    markdown

总结

  1. Markdown 核心语法简洁无门槛，重点掌握标题、列表、代码块、链接 / 图片这几个高频用法就能满足日常编写需求；
  2. 不同编辑器（如 Typora、VS Code、语雀）对 Markdown 的渲染效果略有差异，但核心语法通用；
  3. 文本格式（粗体 / 斜体）、引用、分割线、表格是提升文档可读性的常用补充，按需使用即可。
