windows系统常命令常识

Windows 快捷键与基础命令

出处：微软官方 Windows 10/11 操作系统技术文档（Microsoft Windows Official Documentation），适配 Windows 主流桌面版本。

一、核心快捷键

按通用操作、窗口管理、文件资源管理器、文本编辑、系统功能、截图分类，为高频实用快捷键，无冗余组合。

通用基础操作

  • Ctrl+C：复制选中内容
  • Ctrl+V：粘贴剪贴板内容
  • Ctrl+X：剪切选中内容
  • Ctrl+Z：撤销上一步操作
  • Ctrl+Y：恢复撤销的操作
  • Ctrl+A：全选当前页面 / 文件内容
  • Ctrl+F：打开查找窗口，检索指定内容
  • Ctrl+S：保存当前编辑的文件 / 文档
  • Ctrl+P：打开打印窗口

窗口管理

  • Win+D：快速显示 / 隐藏桌面
  • Win+M：最小化所有打开的窗口
  • Win+Shift+M：还原所有最小化的窗口
  • Win + 左 / 右箭头：将窗口贴靠到屏幕左 / 右半屏
  • Win + 上箭头：最大化当前窗口
  • Win + 下箭头：最小化当前窗口 / 还原最大化窗口
  • Win+Tab：打开任务视图，切换 / 新建虚拟桌面
  • Alt+Tab：循环切换已打开的窗口
  • Alt+F4：关闭当前活动窗口 / 调出关机对话框
  • Ctrl+W：关闭当前浏览器标签 / 软件子窗口

文件资源管理器专用

  • Alt+D：快速定位到地址栏，可直接输入路径
  • Ctrl+Shift+N：快速新建文件夹
  • Ctrl + 滚轮：调整文件 / 文件夹图标大小
  • F2：重命名选中的文件 / 文件夹
  • F5：刷新当前文件夹页面
  • Backspace：返回上一级文件夹

系统功能调用

  • Win+R：打开「运行」对话框，输入命令快速启动程序
  • Win+I：直接打开系统设置
  • Win+E：直接打开文件资源管理器
  • Win+L：快速锁定电脑
  • Ctrl+Alt+Del：调出安全选项界面（锁定 / 任务管理器 / 注销等）
  • Win + 空格：切换输入法
  • Shift + 右键：打开选中对象的高级右键菜单

截图专用

  • Win+Shift+S：打开系统截图工具，支持矩形 / 自由形状 / 窗口 / 全屏截图
  • PrintScreen：截取整个屏幕，截图保存至剪贴板
  • Alt+PrintScreen：截取当前活动窗口，截图保存至剪贴板

二、基础命令

按Win+R 运行命令、CMD/PowerShell 通用命令分类，均为 Windows 原生基础命令，无需额外安装工具，适用于命令提示符（CMD）和 PowerShell。

一、Win+R「运行」快速命令

输入后回车即可直接启动对应系统功能 / 程序，高效快捷：

  • cmd：打开命令提示符（CMD）
  • powershell：打开 PowerShell 终端
  • notepad：打开记事本
  • calc：打开计算器
  • mspaint：打开画图工具
  • control：打开控制面板
  • devmgmt.msc：打开设备管理器
  • services.msc：打开系统服务列表（可启动 / 停止 / 设置服务状态）
  • msconfig：打开系统配置（可设置启动项 / 服务 / 引导参数）
  • regedit：打开注册表编辑器
  • compmgmt.msc：打开计算机管理（含磁盘管理 / 本地用户和组等）
  • diskmgmt.msc：直接打开磁盘管理（分区 / 格式化 / 更改盘符）
  • taskmgr：打开任务管理器
  • explorer：重新启动文件资源管理器（桌面卡死时常用）

二、CMD/PowerShell 通用基础命令

1. 文件 / 目录操作命令

  • cd [路径]：切换当前工作目录，例：cd D:\Software（切换到 D 盘 Software 文件夹）；cd..（返回上一级目录）
  • dir：查看当前目录下的所有文件和文件夹（含详细信息：大小 / 修改时间）
  • md/mkdir [文件夹名]：新建文件夹，例：mkdir Test（新建名为 Test 的文件夹）
  • rd/rmdir [文件夹名]：删除空文件夹，例：rd Test；加参数 /s/q 可删除非空文件夹（rd Test /s/q）
  • copy [源路径] [目标路径]：复制文件，例：copy D:\1.txt E:\
  • move [源路径] [目标路径]：移动文件 / 文件夹，例：move D:\1.txt E:\Test
  • del [文件名 / 后缀]：删除文件，例：del 1.txt；del *.txt（删除当前目录下所有 txt 文件）
  • ren [原文件名] [新文件名]：重命名文件 / 文件夹，例：ren 1.txt 2.docx
  • cls：清空命令行窗口的所有内容，刷新界面

2. 系统信息 / 网络查询命令

  • ipconfig：查看本地网络配置（IP 地址 / 子网掩码 / 网关 / DNS 等）；加参数 /all 可查看详细信息（ipconfig /all）
  • ipconfig /release：释放当前获取的 IP 地址
  • ipconfig /renew：重新获取 IP 地址（DHCP 环境下）
  • systeminfo：查看电脑系统详细信息（系统版本 / 硬件配置 / 补丁等）
  • tasklist：查看当前运行的所有进程（含进程 ID / 占用内存等）
  • ping [IP / 域名]：测试网络连通性，例：ping 8.8.8.8；ping www.baidu.com（加参数 - t 可持续 ping，Ctrl+C 停止）
  • netstat -an：查看当前电脑的所有网络连接（含端口 / 连接状态）

3. 进程 / 系统控制命令

  • taskkill /f/im [进程名]：强制结束指定进程，例：taskkill /f/im notepad.exe（强制关闭记事本）
  • shutdown -s -t [秒数]：定时关机，例：shutdown -s -t 60（60 秒后关机）
  • shutdown -r -t [秒数]：定时重启，例：shutdown -r -t 60
  • shutdown -a：取消所有定时关机 / 重启任务
  • exit：关闭当前 CMD/PowerShell 窗口

4. 其他常用命令

  • attrib：查看 / 修改文件属性，例：attrib +h 1.txt（隐藏 1.txt 文件）；attrib -h 1.txt（取消隐藏）
  • type [文件名]：在命令行窗口查看文本文件内容，例：type 1.txt
