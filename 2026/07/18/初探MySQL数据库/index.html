初探MySQL数据库

初探MySQL数据库

（基于b站博主数解万物教学是视频以及《sql必知必会》）

一.基本概念

  1. 数据（data）：形制各异的外卖
  2. 数据库（database）：一个意义不明的柜子，可以放外卖
  3. 表（table）：外卖隔间，用来个合理存放形制各异外卖，但是要合理分配存储在表里的数据要是统一类型，食品归食品，药品归药品，方便操作
  4. 字段 / 列（column）：隔间里的分类标签，比如外卖编号、收货人、价格、配送时间，规定每条数据拥有哪些属性。
  5. 记录 / 行（row）：单一份完整外卖，一条完整数据，包含所有字段对应的信息。
  6. 主键（primary key）：外卖取餐码，唯一标识每一份外卖，不会重复。
  7. DBMS 数据库管理系统：MySQL 本身，就是管理储物柜、隔间、外卖的整套管理工具，负责增删改查、权限、事务。

二.sql语言分类

1. DDL 数据定义语言

全称：Data Definition Language

作用：定义 / 修改 / 删除数据库、表、索引、视图等库表结构，不操作数据

关键字：CREATE、ALTER、DROP、TRUNCATE、RENAME

2. DQL 数据查询语言

全称：Data Query Language

作用：仅负责查询表数据，开发使用频率最高

关键字：SELECT

3. DML 数据操作语言

全称：Data Manipulation Language

作用：对表内数据增、改、删

关键字：INSERT、UPDATE、DELETE

4. DCL 数据控制语言

全称：Data Control Language

作用：管理数据库账号、分配回收权限，DBA 运维使用

关键字：CREATE USER、DROP USER、GRANT、REVOKE

5. TCL 事务控制语言

全称：Transaction Control Language

作用：管理事务，保证数据一致性，InnoDB 引擎支持

关键字：START TRANSACTION、COMMIT、ROLLBACK、SAVEPOINT、

三.sql语言特点

  1. 关键字不区分大小写
  2. 语句可以多行也可以单行
  3. 以分号结尾
