SQL 数据类型与空间占用

1. 数据类型概念

SQL 数据类型用于规定数据库字段可以存储的数据形式。

选择合适的数据类型可以：

  • 减少存储空间
  • 提高查询效率
  • 限制非法数据
  • 提高数据库设计质量

一、数值类型（Numeric）

1. 整数类型

类型 占用空间 有符号范围 无符号范围 常见用途
TINYINT 1 字节 -128 ~ 127 0 ~ 255 状态、性别、年龄
SMALLINT 2 字节 -32768 ~ 32767 0 ~ 65535 小数量
MEDIUMINT 3 字节 -8388608 ~ 8388607 0 ~ 16777215 中等整数
INT 4 字节 -2147483648 ~ 2147483647 0 ~ 4294967295 普通整数
BIGINT 8 字节 很大范围 很大范围 用户ID、订单号

2. 浮点类型

类型 占用空间 特点 使用场景
FLOAT 4 字节 单精度，存在误差 科学计算
DOUBLE 8 字节 双精度，误差较小 高精度计算
DECIMAL 不固定 精确小数 金额、财务
