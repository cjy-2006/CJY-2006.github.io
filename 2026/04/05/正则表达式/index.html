正则表达式

正则表达式

1.用途

用于提取信息，是对文本（字符串）进行高效、精准的匹配、查找、提取和替换，是处理字符串的 “瑞士军刀”。

2. 核心语法
1. 匹配内容（基础：找 “什么字符”）
语法 核心含义 典型用途
. 匹配任意单个字符（除换行符 \n） 模糊匹配未知字符（如 a.c 匹配 a + 任意字符 + c）
\d 匹配数字（等价 [0-9]） 提取 / 验证数字（手机号、验证码、金额）
\w 匹配字母、数字、下划线（等价 [a-zA-Z0-9_]） 匹配用户名、变量名（如 \w{6,16}）
\s 匹配空白符（空格、Tab、换行） 清理文本中的多余空格
[] 匹配括号内任意一个字符 限定字符范围（如 [a-z] 小写字母、[0-9a-zA-Z] 数字字母）
[^] 匹配括号内除外的任意字符（取反） 排除特定字符（如 [^0-9] 匹配非数字）
2. 限定次数（核心：找 “多少次”）
语法 核心含义 典型用途
+ 匹配前字符 1 次或多次 匹配任意长度数字（如 \d+）
? 匹配前字符 0 次或 1 次 可选字符（如 -?\d+ 匹配正 / 负数字）
{n} 匹配前字符恰好 n 次 固定长度验证（如 \d{11} 匹配 11 位手机号）
{n,} 匹配前字符至少 n 次 最小长度限制（如 \w{6,} 匹配 6 位以上用户名）
{n,m} 匹配前字符 n~m 次 长度范围限制（如 \w{6,16} 匹配 6-16 位密码）
*?/+? 非贪婪匹配（尽可能少） 精准提取（如 a.*?c 从 abc123c 中只取 abc）
3. 锚定位置（关键：限定 “在哪找”）
语法 核心含义 典型用途
^ 匹配字符串开头 验证文本开头（如 ^1\d 匹配以 1 开头的手机号）
$ 匹配字符串结尾 验证文本结尾（如 \d{11}$ 匹配 11 位数字结尾）
^…$ 组合匹配整个字符串 精准验证（如 ^1\d{10}$ 完整验证 11 位手机号）
\b 匹配单词边界 精准匹配独立单词（如 \bcat\b 只匹配 cat，不匹配 category）
4. 组合规则（进阶常用：复杂匹配）
语法 核心含义 典型用途
() 分组（将多个字符视为整体） 对多字符限定次数（如 (abc)+ 匹配 abc/abcabc）
` ` 或运算（匹配两侧任意模式） 匹配多种结果（如 `abc 123匹配abc或123`）
\ 转义符（让元字符变回普通字符） 匹配特殊符号（如 a.c 匹配 a.c，而非 abc）

内置函数和用法

场景 函数 / 方法 核心参数 核心作用 极简示例（可直接运行）
文本验证 re.match(pattern, string, flags=0) pattern：正则规则；string：待匹配文本；flags：匹配模式（如忽略大小写） 从字符串开头匹配，返回匹配对象 / None（需配合^实现全匹配） re.match(r”1\d{10}”, “13812345678”) → 匹配对象；re.match(r”1\d{10}”, “a13812345678”) → None
re.fullmatch(pattern, string, flags=0) 同 match 匹配整个字符串（等价于^pattern$），新手验证优先用 re.fullmatch(r”1\d{10}”, “13812345678”) → 匹配对象；re.fullmatch(r”1\d{10}”, “138123456789”) → None
内容提取 re.findall(pattern, string, flags=0) 同 match 提取所有匹配内容，返回列表（无匹配则为空） re.findall(r”\d+”, “订单123，金额456”) → [‘123’, ‘456’]
re.search(pattern, string, flags=0) 同 match 找到第一个匹配内容，返回匹配对象 / None（需用.group()取值） result = re.search(r”\d+”, “订单123”); result.group() → ‘123’
re.finditer(pattern, string, flags=0) 同 match 提取所有匹配内容，返回迭代器（适合大文本，节省内存） for m in re.finditer(r”\d+”, “订单123”): print(m.group()) → 输出 123
内容替换 re.sub(pattern, repl, string, count=0, flags=0) pattern：匹配规则；repl：替换内容；count：替换次数（0 = 全部） 替换匹配内容，返回新字符串 re.sub(r”垃圾”, “**”, “产品垃圾”) → ‘ 产品 **’；re.sub(r”\d+”, “数字”, “订单123”, count=1) → ‘ 订单数字’
re.subn(pattern, repl, string, count=0, flags=0) 同 sub 替换内容，返回(新字符串, 替换次数)（适合统计替换数量） re.subn(r”\d+”, “数字”, “订单123，金额456”) → (‘ 订单数字，金额数字 ‘, 2)
文本分割 re.split(pattern, string, maxsplit=0, flags=0) pattern：分隔符规则；maxsplit：最大拆分次数（0 = 不限） 按正则规则拆分字符串，返回列表 `re.split(r”\s ,”,” 苹果 香蕉，橙子 “)→[‘ 苹果 ‘, ‘ 香蕉 ‘, ‘ 橙子 ‘]`
进阶用法 re.compile(pattern, flags=0) pattern：正则规则 编译正则（多次复用同一规则时，提升性能） p = re.compile(r”\d+”); p.findall(“订单123”) → [‘123’]
匹配对象.group(n) n：分组编号（0 = 整个匹配，1 = 第一个分组） 提取匹配结果 / 分组内容 r = re.search(r”(\d+)元”, “金额123元”); r.group(1) → ‘123’
匹配对象.span() - 返回匹配内容的起止索引（元组） r = re.search(r”\d+”, “订单123”); r.span() → (2, 5)
