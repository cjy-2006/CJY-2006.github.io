快捷键

  1. Linux 终端技巧与快捷键

    一、历史命令扩展 !

    • !!：执行上一条命令

    • !n：执行第 n 条历史命令

    • !字符串：执行最近以该字符串开头的命令

      • !t → 最近以 t 开头（如 top）
      • !e → 最近以 e 开头（如 echo、exit）

    • !$：用上一条命令的最后一个参数

    • !?字符串?：执行最近包含该字符串的命令

    二、终端核心快捷键

    • Ctrl + C：终止当前进程
    • Ctrl + d：退出登出，等于exit
    • Ctrl + L：清屏（同 clear）
    • Tab：命令 / 路径补全（双击显示全部）
    • Ctrl + A：光标到行首
    • Ctrl + E：光标到行尾
    • ctrl +r 输入内容匹配历史久远的命令，可以左右键修改
    • ctrl + 左右键，左右跳一个单词
    • ↑ / ↓：翻历史命令

    三、实用小技巧

    • 长命令换行：末尾加 \ 回车，命令仍连续
    • history：查看历史命令
    • history | grep 关键词：过滤历史命令
    • man 命令 / 命令 --help：查看帮助
    • 终端复制：Shift+鼠标选中 → Ctrl+Shift+C
    • 终端粘贴：Ctrl+Shift+V

    四、top 命令说明

    • 实时查看系统负载、CPU、内存、进程
    • 输出字段：
      • 系统时间、运行时长、登录用户、负载 1/5/15 分钟
      • 总进程数、运行 / 睡眠 / 停止 / 僵尸进程
      • CPU 占比：us 用户、sy 内核、id 空闲
      • 物理内存与交换分区使用
      • 进程 PID、用户、CPU / 内存占比、命令
