目录结构

linux的目录结构

linux系统目录结构是一个树形结构，与Windows和macOS有所区别的是，linux没有盘符（c盘、d盘），

linux只有根目录（/）所有文件都在根目录下面

linux的顶级目录只有一个根目录

linux的路径表达方式

linux系统中层次关系用：/来表示 例如 /usr/local/1 第一个/是根目录，其他表示层级关系

Windows用：\来表示

home目录是符号~

表示路径

1./test/hello.txt

2./ithema.txt

linux命令基础

什么是命令、命令行

1.命令行：即linux终端（Terminal）。是一种命令提示符页面。以纯字符的形式操作系统

  1. 命令：linux相关程序但是没有图形化页面，输入命令后提供字符反馈

  2. 命令行是命令的载体，finalshell就是一个终端（命令行）软件

linux命令基础格式

通用格式：command [-options] [paramater]

1.command:命令本身

2.-options【可选非必填】命令的选项，用于控制命令的细节

3.paramater：【可选，非必填】命令的参数，用于命令的指向目标ls命令

当前工作目录

在命令行中会默认加载一个home目录作为当前工作目录目录

home目录

每个linux用户在系统中的个人账户目录，路径：/home/用户名

ls命令

1.语法细节:ls [-a -l -h] [linux路径]

2.基本作用：只输入ls会列出自己当前的工作目录下的所有文件夹

ls命令的参数使用

参数是路径，不写参数就列举当前工作目录，输入路径就会列举参数路径内的内容

ls命令的选项使用

选项

  1. -a ：表示all的意思列出所有全部文件（包括隐藏文件/文件夹）

    在linux中以 **’’.’’**开头的文件是隐藏文件

    只有通过-a选项才能看到

  2. -l ：表示以列表（竖向排列）的形式展现内容，并展示更多信息

  3. -h ：表示以易于阅读的方式，列出文件大小如K.,M,G

    h必须要和l或a混合使用，否则无效

选项的· 组合使用

都是同时使用ls -l 和ls -a的效果相同66他

写法

ls -l -a

ls -al

ls -la

选项也可以和参数混合使用
