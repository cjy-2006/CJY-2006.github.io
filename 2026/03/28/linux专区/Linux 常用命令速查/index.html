基础命令速查

Linux 常用命令速查

cd 命令（切换目录）

  1. 语法：cd [Linux目录]
    • cd 无需选项，只有参数，表示进入目标目录
  2. 常用用法：
    • cd（无参数）：进入当前用户的 home 目录
    • cd -：退回上一次所在的目录
    • cd ..：回到上一级目录
    • cd ../..：连续退回两级目录

特殊路径符

  1. . ：表示当前目录（例：cd ./desktop 与 cd desktop 效果一致）
  2. .. ：表示上一级目录
  3. ~ ：表示 home 目录

相对路径和绝对路径

  1. 绝对路径：以根目录 / 为起点（例：/home/user/desktop）
  2. 相对路径：以当前工作目录为起点（例：./desktop）

pwd 命令（查看当前目录）

  1. 无参数、无选项
  2. 功能：输出当前工作目录（print work directory）

mkdir 命令（创建目录）

  1. 功能：创建新目录（make directory）
  2. 语法：mkdir [-p] Linux路径
    • 参数：必填，目标文件夹路径（相对/绝对均可）
    • -p 选项：创建不存在的父目录（例：mkdir -p test/test1 会同时创建 test 和 test1）
  3. 注意：普通用户需在 home 目录内创建，否则会因权限不足失败

touch 命令（创建文件）

  1. 功能：创建空文件
  2. 语法：touch Linux路径
  3. 说明：无选项，路径支持相对/绝对/特殊路径符；ls -l 结果中，d 表示文件夹，- 表示文件

cat 命令（查看文件内容）

  1. 功能：一次性显示文件全部内容
  2. 语法：cat 路径

more 命令（分页查看文件内容）

  1. 功能：分页查看大文件内容
  2. 语法：more 路径
  3. 操作：空格翻页，q 退出

cp 命令（复制）

  1. 功能：复制文件/文件夹（copy）
  2. 语法：cp [-r] 参数1 参数2
    • 参数1：被复制的文件/文件夹路径
    • 参数2：目标路径
    • -r 选项：复制文件夹时必须加（递归复制所有子内容）

mv 命令（移动/重命名）

  1. 功能：移动文件/文件夹，或重命名
  2. 语法：mv 参数1 参数2
    • 参数1：被移动的文件/文件夹路径
    • 参数2：目标路径；若目标不存在则为重命名（需确保目标父目录存在）

rm 命令（删除）

  1. 功能：删除文件/文件夹（remove）
  2. 语法：rm [-r -f] 参数1 [参数2 …]
    • 参数：待删除文件/文件夹的路径（可多个）
    • -r：删除文件夹时必须加（递归删除）
    • -f：强制删除（普通用户无提示，root 用户慎用）
  3. 通配符支持：
    • *test：匹配以 test 结尾的内容
    • test*：匹配以 test 开头的内容
    • test：匹配包含 test 的内容
  4. 危险命令：
    • rm -rf / 或 rm -rf /*：删除根目录所有内容，等同于格式化系统，严禁执行

root 用户（超级管理员）

  1. Linux 中权限最高的用户，UID=0
  2. 拥有系统全部权限：删除系统文件、修改任意配置、启停所有服务

su 命令（切换用户）

  1. 全称：switch user（切换用户）
  2. 功能：从当前用户切换到另一个用户（最常用切换到 root）
  3. 说明：是命令工具，不是用户

which 命令（查找命令路径）

  1. 功能：查看命令程序文件的绝对路径
  2. 语法：which 要查找的命令
  3. 补充：which cd 无输出的原因
    • which 仅查找 PATH 环境变量中的外部可执行程序
    • cd 是 Shell 内置命令（内嵌在 Shell 解释器中），无独立文件路径，因此无法找到

find 命令（搜索文件）

  1. 功能：按条件搜索指定文件（支持通配符）
  2. 语法：
    • 按文件名查找：find 起始路径 -name ‘文件名’
    • 按大小查找：find 起始路径 -size +|-n[kMG]
      • +/-：表示大于/小于
      • n：数字（大小值）
      • k（小写）：KB；M：MB；G：GB
  3. 常见问题：find 找到 /proc 下的文件后，ls 提示不存在
    • /proc 是动态虚拟文件系统，存储进程实时状态
    • 进程结束后，对应的 /proc/[PID] 目录会立即删除，因此 ls 时路径已不存在

grep 命令（过滤内容）

  1. 功能：按关键字过滤文件行（不支持通配符）
  2. 语法：grep [-n] ‘关键字’ 目标文件路径
    • -n 选项：显示关键字所在行号
    • 目标路径：支持作为内容输入端口（配合管道符使用）

wc 命令（统计内容）

  1. 功能：统计文件的行数、单词数、字节数等
  2. 语法：wc [-c -m -l -w] 文件路径
    • -c：统计字节数
    • -m：统计字符数
    • -l：统计行数
    • -w：统计单词数

管道符 |

  1. 功能：将左侧命令的输出作为右侧命令的输入
  2. 语法：命令1 | 命令2
  3. 要求：命令1 必须有输出内容（例：ls | grep test）

echo 命令（输出内容）

  1. 功能：在命令行输出指定内容
  2. 语法：echo ‘输出内容’

反引号 ``

  1. 功能：执行反引号内的命令，而非当作普通字符
  2. 示例：echo pwd 会输出当前目录路径（先执行 pwd，再把结果传给 echo）

重定向符 > 和 >>

  1. ：将左侧命令结果覆盖写入右侧文件（文件不存在则创建）

  2. ：将左侧命令结果追加写入右侧文件（文件不存在则创建）

tail 命令（查看文件尾部内容）

  1. 功能：查看文件尾部内容，支持实时跟踪文件更新
  2. 语法：tail [-f -num] 路径
    • -f：持续追踪文件最新内容（实时刷新）
    • -num：指定查看的行数（默认 10 行，例：tail -20 文件名 查看最后 20 行）
